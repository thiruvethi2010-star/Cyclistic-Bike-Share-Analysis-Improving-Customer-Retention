# Cyclistic Bike-Share Analysis: Improving Customer Retention

## Project Overview
This project analyzes Cyclistic bike-share data to understand user behavior and support marketing strategies.

## Business Problem
The goal is to identify differences between casual riders and annual members to help increase customer conversion.

## Tools Used
- Excel
- SQL
- Tableau

## Process
- Data cleaning and preparation
- Data analysis
- Data visualization

## Key Insights
- Casual riders use bikes more on weekends
- Members use bikes more frequently during weekdays
- Behavioral differences can guide marketing strategies

## Conclusion
The analysis provides insights that can help Cyclistic improve customer retention and convert casual users into members.
